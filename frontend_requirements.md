# 前端 Node.js 依赖

## 安装方法
1. 进入前端目录：`cd front`
2. 安装依赖：`npm install`

## 依赖列表

### 生产依赖
- vue: ^3.5.24

### 开发依赖
- @vitejs/plugin-vue: ^5.0.4
- electron: ^27.0.0
- electron-builder: ^24.12.0
- vite: ^5.2.0

## 脚本命令
- `npm run dev`: 启动开发服务器
- `npm run build`: 构建生产版本
- `npm run preview`: 预览构建结果
- `npm run electron:dev`: 启动 Electron 开发模式
- `npm run electron:build`: 构建 Electron 应用
