# 每周礼包领取脚本使用说明

## 功能介绍

本脚本用于自动领取《崩坏3》游戏中的每周免费礼包。脚本会自动：

1. 聚焦 BH3 游戏窗口
2. 点击商城
3. 点击礼包
4. 点击周期
5. 点击免费领取礼包
6. 再次点击免费确认领取
7. 按两次 ESC 返回主界面

## 依赖环境

### 1. 系统要求
- Windows 操作系统
- 需要以管理员身份运行

### 2. 游戏设置要求
- 游戏分辨率：建议设置为 1920x1080 全屏模式
- 舰桥场景：建议使用「永恒的礼堂」场景，以确保点击位置准确
- 游戏语言：简体中文

### 3. 依赖文件
脚本依赖以下文件和模块：
- `on_window.py` - 窗口操作模块
- `photos/clicks_keyboard.py` - 点击和键盘操作模块
- `process/main_screen.py` - 主界面相关操作
- `all_log/save_output.py` - 日志输出模块
- `photos/` 目录下的相关截图文件

## 使用方法

### 方法一：直接运行

1. 确保游戏已启动并登录到主界面
2. 右键点击 `everyweek_gift.py` 文件
3. 选择「以管理员身份运行」
4. 观察控制台输出，等待脚本执行完成

### 方法二：通过命令行运行

1. 打开命令提示符（管理员模式）
2. 切换到脚本所在目录：
   ```cmd
   cd tools
   ```
3. 运行脚本：
   ```cmd
   python everyweek_gift.py
   ```

## 注意事项

1. **权限要求**：脚本必须以管理员身份运行，否则可能无法正常操作游戏窗口
2. **游戏状态**：运行脚本前，请确保游戏已启动并处于主界面状态
3. **界面要求**：请确保游戏界面清晰可见，无其他窗口遮挡
4. **网络环境**：确保网络连接正常，避免领取过程中网络中断
5. **安全提示**：本脚本仅用于辅助操作，不会修改游戏数据，请放心使用

## 常见问题

### 1. 脚本提示「BH3窗口聚焦失败」
- 检查游戏是否已启动
- 确保游戏窗口标题包含「崩坏3」字样

### 2. 脚本提示「商城点击失败」
- 尝试切换舰桥场景为「永恒的礼堂」
- 调整游戏分辨率为 1920x1080 全屏

### 3. 脚本执行过程中卡住
- 可能是游戏界面加载较慢，耐心等待
- 或尝试手动干预后重新运行脚本

## 故障排除

如果脚本执行失败，请检查以下几点：

1. 检查所有依赖文件是否存在
2. 确认游戏设置是否符合要求
3. 尝试重启游戏后重新运行脚本
4. 检查是否有其他程序干扰游戏窗口

## 脚本执行流程

1. 检查管理员权限
2. 聚焦 BH3 游戏窗口
3. 点击商城按钮
4. 点击礼包选项
5. 点击周期选项
6. 点击免费按钮领取礼包
7. 再次点击免费按钮确认
8. 按 ESC 键返回主界面
9. 确保回到游戏主界面
