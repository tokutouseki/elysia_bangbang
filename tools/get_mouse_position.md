# get_mouse_position.py 使用说明

## 功能描述
该脚本用于记录鼠标点击位置和矩形选择范围，主要用于获取游戏中的坐标点，方便后续自动化操作。

## 安装依赖
```bash
pip install pyautogui pynput
```

## 使用方法
1. 以管理员身份运行脚本：
   - 右键点击脚本文件
   - 选择"以管理员身份运行"

2. 脚本会自动聚焦到BH3游戏窗口

3. 执行单次点击记录：
   - 脚本运行后会等待用户点击
   - 点击鼠标左键，脚本会输出点击坐标

4. 执行矩形选择：
   - 调用`record_two_clicks_and_show_rectangle()`函数
   - 脚本会等待用户执行两次左键点击
   - 输出两次点击的坐标和计算出的矩形范围

## 主要函数

### click_and_print_position()
- **功能**：记录单次鼠标点击位置
- **返回值**：成功返回True，失败返回False

### record_two_clicks_and_show_rectangle()
- **功能**：记录两次鼠标点击并计算矩形范围
- **返回值**：成功返回(left, top, right, bottom)元组，失败返回None

## 注意事项
1. 必须以管理员身份运行，否则可能无法正常工作
2. 脚本会自动聚焦到BH3游戏窗口，确保游戏窗口已经打开
3. 按Ctrl+C可以退出监听状态
4. 坐标值基于屏幕分辨率，不同分辨率下坐标会有所不同
