# 舰团贡献脚本使用说明

## 功能介绍
本脚本用于自动执行崩坏3游戏中的舰团贡献操作，具体功能如下：
- 自动检查并提权为管理员权限
- 自动聚焦BH3游戏窗口
- 自动点击"舰团"按钮
- 自动点击"舰团贡献"按钮
- 自动点击"舰团贡献_5000"按钮
- 自动返回游戏主界面

## 使用要求
1. **管理员权限**：脚本需要以管理员身份运行才能正常工作
2. **游戏状态**：需要确保崩坏3游戏已打开并处于主界面
3. **分辨率要求**：建议使用1980*1080分辨率全屏模式
4. **舰桥要求**：建议使用"永恒的礼堂"舰桥

## 使用方法
1. 确保崩坏3游戏已启动并处于主界面
2. 右键点击本脚本文件，选择"以管理员身份运行"
3. 脚本会自动执行以下操作：
   - 检查管理员权限并尝试自动提权
   - 聚焦游戏窗口
   - 执行舰团贡献操作
   - 返回主界面
4. 操作完成后，脚本会在控制台输出详细的执行日志

## 执行流程
1. 检查管理员权限
2. 聚焦BH3窗口
3. 点击"舰团"按钮
4. 点击"舰团贡献"按钮
5. 点击"舰团贡献_5000"按钮
6. 按下ESC键三次返回主界面
7. 执行make_on_main()函数确保回到主界面

## 注意事项
1. 脚本执行过程中请勿操作鼠标和键盘
2. 如果游戏界面与脚本预期不符，可能会导致操作失败
3. 若出现操作失败，脚本会在控制台输出失败信息
4. 若自动提权失败，请手动右键点击脚本并选择"以管理员身份运行"

## 常见问题
- **问题**：脚本无法聚焦游戏窗口
  **解决方法**：确保游戏已打开并处于前台，或手动点击游戏窗口

- **问题**：点击按钮失败
  **解决方法**：检查游戏分辨率是否为1980*1080全屏，舰桥是否为"永恒的礼堂"

- **问题**：管理员权限提权失败
  **解决方法**：手动右键点击脚本，选择"以管理员身份运行"
