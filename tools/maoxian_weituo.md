# 冒险委托脚本使用说明

## 功能介绍
本脚本用于自动执行崩坏3游戏中的冒险委托操作，具体功能如下：
- 自动检查并提权为管理员权限
- 自动聚焦BH3游戏窗口
- 自动点击"出击"按钮
- 自动点击"出击_出击"按钮
- 自动点击"开放世界"按钮
- 自动点击"地图_选择"按钮
- 自动点击"后崩坏书1"按钮
- 自动点击"地图_进入"按钮
- 自动点击"后崩坏书1_委托"按钮
- 自动点击"委托开启"按钮
- 自动点击指定位置(1001, 859)
- 自动按下并释放"i"键
- 自动执行委托选择和接取操作（重复5次）
- 自动点击"委托减负"按钮
- 自动点击"确定"按钮
- 自动返回游戏主界面

## 使用要求
1. **管理员权限**：脚本需要以管理员身份运行才能正常工作
2. **游戏状态**：需要确保崩坏3游戏已打开并处于主界面
3. **分辨率要求**：建议使用1980*1080分辨率全屏模式
4. **界面要求**：游戏界面需要与脚本预期的界面布局一致

## 使用方法
1. 确保崩坏3游戏已启动并处于主界面
2. 右键点击本脚本文件，选择"以管理员身份运行"
3. 脚本会自动执行以下操作：
   - 检查管理员权限并尝试自动提权
   - 聚焦游戏窗口
   - 执行冒险委托的完整操作流程
   - 返回主界面
4. 操作完成后，脚本会在控制台输出详细的执行日志

## 执行流程
1. 检查管理员权限
2. 聚焦BH3窗口
3. 点击"出击"按钮
4. 点击"出击_出击"按钮
5. 点击"开放世界"按钮
6. 点击"地图_选择"按钮
7. 点击"后崩坏书1"按钮
8. 点击"地图_进入"按钮
9. 点击"后崩坏书1_委托"按钮
10. 点击"委托开启"按钮（失败后会重试一次）
11. 点击位置(1001, 859)
12. 按下并释放"i"键
13. 再次点击"委托开启"按钮
14. 重复执行5次以下操作：
    - 点击"委托选择"按钮
    - 点击"委托接取"按钮
    - 按下并释放"i"键
15. 点击"委托减负"按钮
16. 点击"确定"按钮
17. 按下ESC键四次返回主界面
18. 执行make_on_main()函数确保回到主界面

## 注意事项
1. 脚本执行过程中请勿操作鼠标和键盘
2. 如果游戏界面与脚本预期不符，可能会导致操作失败
3. 若出现操作失败，脚本会在控制台输出失败信息
4. 若自动提权失败，请手动右键点击脚本并选择"以管理员身份运行"
5. 脚本执行过程中会有多次等待，请耐心等待执行完成

## 常见问题
- **问题**：脚本无法聚焦游戏窗口
  **解决方法**：确保游戏已打开并处于前台，或手动点击游戏窗口

- **问题**：点击按钮失败
  **解决方法**：检查游戏分辨率是否为1980*1080全屏，确保游戏界面与脚本预期一致

- **问题**：管理员权限提权失败
  **解决方法**：手动右键点击脚本，选择"以管理员身份运行"

- **问题**：委托操作未完成
  **解决方法**：确保游戏中已有可接取的委托任务，且角色等级满足委托要求
