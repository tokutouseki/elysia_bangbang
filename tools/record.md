# record.py 使用说明

## 功能描述
该脚本用于记录用户的键盘和鼠标操作，生成JSON格式的操作记录文件，方便后续通过replay_keyboard.py脚本复现操作。

## 安装依赖
```bash
pip install pynput
```

## 使用方法
1. 以管理员身份运行脚本：
   - 右键点击脚本文件
   - 选择"以管理员身份运行"

2. 脚本会自动聚焦到BH3游戏窗口

3. 开始记录操作：
   - 脚本启动后会自动开始记录
   - 执行需要记录的操作（键盘按键和鼠标点击）

4. 停止记录：
   - 按Ctrl+C停止记录
   - 脚本会自动保存记录到reproduce目录

## 记录内容
- **键盘事件**：记录指定按键的按下和松开时间
  - 支持的按键：w, a, s, d, j, k, l, u, i, f, 1, 2

- **鼠标事件**：记录鼠标左键点击和滚轮滚动
  - 左键按下和松开的坐标及时间
  - 滚轮滚动的方向和位置

## 输出文件
- **文件位置**：`tools/reproduce/`目录
- **文件命名**：`recording_cn_时间戳.json`
- **文件格式**：JSON格式，包含以下字段：
  - 开始时间
  - 结束时间
  - 持续时长
  - 键盘事件列表
  - 鼠标事件列表

## 主要类和方法

### OperationRecorder 类
- **初始化**：创建记录器实例，设置输出文件路径

- **start()**：开始记录操作
  - **返回值**：成功返回True，失败返回False

- **stop()**：停止记录操作并保存数据
  - **返回值**：成功返回True，失败返回False

## 注意事项
1. 必须以管理员身份运行，否则可能无法正常工作
2. 脚本会自动聚焦到BH3游戏窗口，确保游戏窗口已经打开
3. 只记录指定的按键，其他按键不会被记录
4. 记录文件会保存在reproduce目录，确保该目录存在
